### node

[官网](https://nodejs.org/en/)，或者[中文社区](http://nodejs.cn)下载安装包，安装成功后
```
This package has installed:
•    Node.js v8.9.2 to /usr/local/bin/node
•    npm v5.5.1 to /usr/local/bin/npm
Make sure that /usr/local/bin is in your $PATH.
```
显示安装了Node和包管理器npm
